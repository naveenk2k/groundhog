## Agent skills

### Issue tracker

Issues live as GitHub issues on `naveenk2k/groundhog`, via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default five-role vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) — none exist as labels yet. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo: `CONTEXT.md` is the domain glossary, `DECISIONS.md` plays the `docs/adr/` role. See `docs/agents/domain.md`.
